Subprojects

psyplot is only the over-arching framework. It’s capabilities are splitted into several subprojects. Each of them is accessible via https://psyplot.github.io/<project-name>

  • the psyplot_gui package: The GUI to psyplot Source

  • the psy_view package: An ncview-like interface for psyplot Source

  • the psy-simple package: A plugin for simple visualization Source

  • the psy-maps package: A psyplot plugin for visualizing data on a map Source

  • the psy-reg package: A psyplot plugin for visualizing and calculating regression fits Source

See Psyplot plugins for more informations on the plugins.
